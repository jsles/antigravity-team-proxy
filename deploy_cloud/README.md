# Antigravity Manager Cloud Deployment

이 Space는 팀 공용 Antigravity-Manager 프록시 서버입니다.

### 설정 방법 (Settings -> Variables)
1. `API_KEY`: 팀원들이 사용할 `sk-antigravity-team-shared` 또는 고유 키
2. `WEB_PASSWORD`: 관리 대시보드 로그인 비밀번호
